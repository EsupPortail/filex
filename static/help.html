1 Qu'est-ce que FileX ?

FileX est un système de transfert de fichier.

Il a été conçu dans le but de décharger la messagerie électronique de cette tâche. En effet, nombre d'utilisateurs de messagerie utilisent la fonctionnalité des "pièces jointes" pour transmettre des documents volumineux, ce qui occasionne quelques désagréments pour les destinataires :

  • Boîte de mail saturée
  • Impossibilité de recevoir les messages s'ils sont trop volumineux
  • Lenteur de réception des mails

Le principe de fonctionnement du système est le suivant :

  • Vous déposez un fichier
  • Une adresse internet identifiant le fichier est créée
  • Vous envoyez cette adresse par email au(x) destinataire(s) du fichier
  • Le(s) destinataire(s) récupère(nt) le fichier en utilisant son (leur) navigateur internet

Le seul outil nécessaire pour effectuer toutes ces opérations est le navigateur internet !

2 Déposer un fichier

Pour déposer un fichier : sélectionnez un fichier depuis votre disque dur en utilisant le bouton parcourir..., et appuyez sur le bouton envoyer.

Une fenêtre indiquant la progression du chargement du fichier sur le serveur apparaîtra.

Il est possible que cette fenêtre ne soit pas visible, selon la taille du fichier déposé ou le navigateur utilisé.

Lorsque la dépose est terminée, un message vous indiquera l'adresse à laquelle le fichier pourra être téléchargé durant sa période de validité. Vous recevrez également une confirmation de dépose de fichier par courrier électronique.

Pour qu'une personne récupère votre fichier, faites lui passer cette adresse par courrier électronique ou tout autre moyen de communication.

2.1 Les options de dépose

Les options suivantes sont disponibles lors de la dépose d'un fichier :

  • Expiration du fichier

    Le fichier ne sera plus disponible au téléchargement après le nombre de jours sélectionné.

  • Recevoir un avis de réception à chaque téléchargement

    Recevoir un e-mail de confirmation à chaque téléchargement du fichier.

  • Recevoir un récapitulatif des téléchargements lorsque le fichier aura expiré

    Recevoir un e-mail détaillant les téléchargements du fichier lorsque celui-ci arrivera à expiration.

  • Utiliser un mot de passe pour le téléchargement

    Vous pouvez définir un mot de passe qui sera demandé à chaque téléchargement de votre fichier.

    Pensez à faire parvenir le mot de passe aux personnes devant télécharger votre fichier.

Toutes ces options sont modifiables ultérieurement via la Gestion de vos fichiers.

3 Gestion de vos fichiers

La Gestion de vos fichiers est accessible depuis la page de dépose de fichier. Elle permet de consulter l'historique de tous les fichiers que vous avez déposés sur le système et d'en modifier certaines options.

La liste de vos fichiers est triée par défaut de manière décroissante sur la date de dépose (le fichier le plus récent en haut).

Vous pouvez trier la liste de vos fichiers en cliquant sur + (ordre croissant) et - (ordre décroissant) dans les en-têtes de colonne.

Pour avoir des informations détaillées sur un fichier et modifier certaines options, cliquez sur le nom du fichier.

3.1 Modification des options de vos fichiers

Les options suivantes sont modifiables si le fichier n'a pas expiré :

  • Mot de passe

    Cette option vous permet de définir (ou supprimer) un mot de passe pour le téléchargement de votre fichier. Sélectionnez non pour ne pas avoir à utiliser de mot de passe pour télécharger votre fichier ou oui et saisissez un mot de passe dans le champ de droite ; soit pour créer un nouveau mot de passe, soit pour mettre à jour un mot de passe précédemment défini.

    N'oubliez pas de faire parvenir ce mot de passe aux personnes devant télécharger votre fichier

  • Augmenter la durée de vie du fichier

    Cette option, si elle est active ou si vous n'avez pas dépassé le nombre de renouvellements permis, vous permet de prolonger la durée de vie de votre fichier de n jours.

  • Expiré

    Sélectionnez oui si vous voulez faire expirer le fichier. Il ne sera alors plus disponible au téléchargement et sera supprimé du système. Cette action est définitive.

  • Recevoir un message à chaque téléchargement

    Sélectionnez oui ou non pour recevoir ou ne plus recevoir l'e-mail de confirmation de téléchargement de fichier.

  • Recevoir un récapitulatif lorsque le fichier sera supprimé

    Sélectionnez oui ou non pour recevoir un e-mail détaillant les téléchargements du fichier lorsque celui-ci arrivera à expiration.

Cliquez sur le bouton "Valider" pour appliquer vos modifications.

5 FAQ

5.1 Je veux déposer plusieurs fichiers

Il n'est possible de déposer qu'un seul fichier à la fois, aussi, si vous désirez déposer plusieurs fichiers qui doivent être téléchargés simultanément, regroupez les en utilisant des utilitaires tels que zip, rar ...

5.2 Je ne vois pas la fenêtre de progression de dépose de fichier

Il y a deux possibilités à ce "problème" :

  • le chargement du fichier est trop rapide, la fenêtre se ferme automatiquement lorsque le chargement est terminé.
  • la fenêtre de progression ne fonctionne que sur des navigateurs de génération supérieure à 4 pour Netscape et Internet Explorer, ainsi que opera, mozilla (et dérivés : phoenix, firebird, galeon ...), konqueror

Quoi qu'il en soit, c'est le message de confirmation de dépose de fichier qui vous indiquera que celui-ci est bien déposé.

5.3 Je ne me rappelle plus de l'adresse de téléchargement de mon fichier

Retournez sur la page de dépose de fichier et suivez le lien Gestion de vos fichiers. Localisez votre fichier et cliquez sur son nom. Vous obtiendrez ainsi des informations concernant votre fichier ainsi que son adresse de téléchargement si celui-ci n'a pas expiré.
